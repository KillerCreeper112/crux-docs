# GETTING STARTED

CruxEnchantsOverhaul changes the Minecraft vanilla enchanting table and anvil mechanics.

## INSTALLATION REQUIREMENTS
- **Java 25** or <span style={{ color: 'green' }}>newer</span>
- Supported Versions: **26.1**, **1.21.11** [(Paper)](https://papermc.io/software/paper/)
